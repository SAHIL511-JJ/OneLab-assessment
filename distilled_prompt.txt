Distilled Prompt

Build a payment reconciliation tool with a simple dashboard. Compare transactions vs bank settlements by reference ID, calculate expected vs actual amounts, and classify rows as matched, out-of-tolerance mismatch, missing in bank, or unexpected in bank. Use a configurable tolerance (for example, INR 10) to suppress small per-row exceptions, but still show true total financial variance (short, over, net). Include an exceptions view, summary metrics, CSV upload with column mapping and validation, JSON report output, and basic tests.
